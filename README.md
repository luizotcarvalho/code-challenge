#Pre-requisitos
* `node` <= 0.12.x
* `gulp-cli` instalado globalmente

#Setup
Instala as dependências do projeto.
```
npm install
bower install
```

#Rodando
Abre o navegador no `localhost:3002` acessando o `src` da aplicação.
```
gulp serve
```

#Testes
Roda alguns testes unitários.
```
gulp test
```

#Build
Gera a pasta `dist` com os scripts minificados.
```
gulp build
```
